# Glossary

## Artifact
A document or file used as input to the model.

## Context
The complete set of inputs provided to the model, including:
- artifacts
- conversation history
- system prompt

## Retrieval
The process of selecting which artifacts are included in context.

## Session
A project-scoped conversation stored on disk.

## Message
A single unit of communication within a session.

## Global Documents
Documents that apply across all projects.

## Project Documents
Documents specific to a single project.

## Retrieval Policy
Rules that determine which artifacts are included in context.

## Core Engine
The system responsible for:
- context construction
- session handling
- model interaction

## Surface
A user interface layer (CLI, PowerShell, browser, editor).

## Prompt
The structured input sent to the model.

## Summary
A condensed representation of prior conversation history.

## Intent
The classification of a request (planning, coding, review, general).
