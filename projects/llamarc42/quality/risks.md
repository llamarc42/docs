# Risks

## Technical Risks

- context size explosion
- inconsistent retrieval behavior
- divergence between surfaces
- over-reliance on model reasoning

## Product Risks

- unclear user workflows
- over-engineering early
- lack of adoption

## Architectural Risks

- logic leaking into multiple layers
- API becoming a second core
- poor boundary enforcement

## Mitigation Strategies

- strict adherence to core engine ownership
- retrieval policy enforcement
- incremental development
