# Tradeoffs

## PowerShell vs C#

- PowerShell: fast iteration
- C#: maintainability and structure

Decision: PowerShell for prototype, C# for core

---

## Retrieval vs Fine-Tuning

- Retrieval: current, flexible
- Fine-tuning: static, brittle

Decision: retrieval-first

---

## Local vs Cloud

- Local: control, privacy
- Cloud: capability

Decision: local-first

---

## Simplicity vs Scalability

- Simplicity: easier iteration
- Scalability: long-term flexibility

Decision: start simple, evolve as needed
